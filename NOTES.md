# CampusOps Agent Notes

## Current MVP
- Event idea input
- Gemini Planning Agent generates clarifying questions
- User answers questions
- Gemini Plan Generator Agent creates summary, timeline, tasks, and risks
- Gemini Critic Agent reviews plan
- Gemini Drafting Agent creates event announcement
- Agent Activity log shows each agent run
